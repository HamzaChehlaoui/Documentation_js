Documentation JavaScript

Revoir les bases de JavaScript

Variables

Les variables sont des conteneurs pour stocker des valeurs. JavaScript utilise les mots-clés let, const, et var pour déclarer des variables.

Types de Données

Les types primitifs en JavaScript sont :

  • String (ex : "Bonjour Monde")
  • Number (ex : 42, 3.14)
  • Boolean (ex : true, false)
  • Null
  • Undefined

Opérateurs

Les opérateurs en JavaScript incluent les opérateurs arithmétiques, de comparaison et logiques :

  • Opérateurs arithmétiques : +, -, *, /, etc.
  • Opérateurs de comparaison : ==, ===, !=, etc.
  • Opérateurs logiques : &&, ||, !

Structures de Contrôle

Les structures de contrôle incluent les conditions (if, switch) et les boucles (for, while).

Apprendre à manipuler le DOM

Sélectionner des Éléments

Utilisez JavaScript pour manipuler le DOM. Par exemple, pour sélectionner un élément par son identifiant : document.getElementById("id")

Modifier des Éléments

Vous pouvez modifier des éléments, par exemple en changeant leur contenu HTML avec innerHTML ou textContent.

Ajouter et Supprimer des Éléments

Utilisez document.createElement pour créer de nouveaux éléments, puis ajoutez-les avec appendChild ou insertBefore.

Événements

Les événements sont essentiels pour interagir avec les utilisateurs. Utilisez addEventListener pour écouter les événements.

          // Exemple de calculatrice simple
          function calculator() {
            const num1 = prompt("Entrez le premier nombre:");
            const num2 = prompt("Entrez le deuxième nombre:");
            const operation = prompt("Entrez l'opération (+, -, *, /):");

            let result;
            if (operation === "+") {
              result = parseFloat(num1) + parseFloat(num2);
            } else if (operation === "-") {
              result = parseFloat(num1) - parseFloat(num2);
            } else if (operation === "*") {
              result = parseFloat(num1) * parseFloat(num2);
            } else if (operation === "/") {
              result = parseFloat(num1) / parseFloat(num2);
            }
          }
